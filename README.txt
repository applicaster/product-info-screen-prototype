Responsive Grid Template for info-screens
=========================================

The entry point to this project is via the index.html file.
In the file there is an HTML list that contains 6 place holder items. Please change these items with your real content.
Each item contains an image (640px * 360px) and caption text that you can change.

Serving the files from a web server
===================================
Please make sure you serve all the files & folders in this directory and not only the index.html
